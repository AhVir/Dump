content-box vs border-box

Both divs have: width: 200px; padding: 20px; border: 5px solid black;

content-box

Total width becomes 250px.

Text is NOT pushed inward.
Padding + border increase the box size OUTSIDE.
border-box

Total width stays 200px.

✔️ Text is pushed inward.
Padding + border fit INSIDE the set width.
